Cara Mudah Menginstal Aplikasi

Panduan pemasangan PesisirKita secara manual (Sideloading) di perangkat Android Anda.

1

Unduh File APK

Tekan tombol "Unduh APK Resmi" di bagian atas halaman ini. Jika muncul peringatan keamanan dari browser (Chrome/Edge), pilih "Tetap Download" (Download anyway). File ini aman dan resmi dari repositori kami.

File: pesisirkita.apk
2

Izinkan Akses Pemasangan

Buka file APK yang sudah selesai diunduh. Sistem Android biasanya akan memblokir instalasi dari luar Play Store. Pada pop-up yang muncul, klik Pengaturan (Settings) lalu aktifkan tombol "Izinkan dari sumber ini" (Allow from this source).

Aktifkan toggle perizinan
3

Pasang dan Jalankan

Setelah izin diberikan, tekan tombol kembali (back) dan klik Instal (Install). Tunggu beberapa detik, dan aplikasi PesisirKita siap digunakan di lapangan. Selamat mencoba!

Instrumen & Evaluasi Data

Dikembangkan berdasarkan kerangka kerja UNESCO Sandwatch dan metode observasi lapangan tersertifikasi.

Pencatatan Lapangan (Field)

Sistem formulir cerdas untuk pencatatan koordinat GPS, pembuatan poligon area luasan (Ha), dan lampiran bukti visual. Otomatis menyimpan secara lokal di HP tanpa sinyal.

Kalkulator Ekologi

Instrumen standar untuk mengubah data lapangan menjadi ukuran metrik dampak nyata. Hitung serapan Karbon Biru mangrove, estimasi Biomassa Karang, dan proyeksi waktu urai limbah pesisir.

Glosarium Saku

Ensiklopedia terstruktur berisi panduan identifikasi taksonomi flora/fauna pesisir, konsep mitigasi bencana, serta indikator kesehatan lingkungan untuk memandu verifikasi lapangan.

Kanvas Ide (Hackathon)

Fasilitas *brainstorming* kolaboratif bagi instansi atau sekolah untuk merumuskan akar permasalahan pesisir dan merancang kerangka solusi inovatif berbasis kerangka *Design Thinking*.
